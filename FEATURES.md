# Features List

Lucky Draw 專案功能規劃與實作狀態追蹤。

## 核心功能

### ✅ 已完成
- [ ] 基礎專案架構

### 🚧 進行中
- [ ] 無

### 📋 待開發

#### Phase 1: 基礎功能
- [ ] **參與者管理**
  - [ ] 手動輸入參與者名單（每行一個名字）
  - [ ] 顯示參與者總數
  - [ ] 清空名單功能
  - [ ] 名單驗證（去除空白、重複檢查）

- [ ] **基礎抽獎功能**
  - [ ] 單次抽獎（抽取一位）
  - [ ] 顯示抽獎結果
  - [ ] 重置抽獎功能

- [ ] **UI/UX**
  - [ ] 響應式設計（支援手機、平板、桌面）
  - [ ] 基礎樣式與佈局
  - [ ] 清晰的操作提示

#### Phase 2: 進階功能
- [ ] **抽獎設定**
  - [ ] 多次抽取設定（一次抽多位）
  - [ ] 重複抽取選項（允許/不允許同一人重複中獎）
  - [ ] 自訂抽獎數量

- [ ] **動畫效果**
  - [ ] 抽獎過程動畫
  - [ ] 名單滾動效果
  - [ ] 結果顯示動畫
  - [ ] 音效選項（可選）

- [ ] **結果管理**
  - [ ] 抽獎歷史記錄
  - [ ] 已抽中名單顯示
  - [ ] 剩餘名單顯示
  - [ ] 匯出結果功能

#### Phase 3: 資料管理
- [ ] **匯入/匯出**
  - [ ] CSV 檔案匯入
  - [ ] JSON 檔案匯入
  - [ ] 文字檔案匯入
  - [ ] 匯出抽獎結果（CSV/TXT）
  - [ ] 匯出完整記錄（JSON）

- [ ] **本地儲存**
  - [ ] 使用 LocalStorage 儲存名單
  - [ ] 儲存抽獎歷史
  - [ ] 儲存使用者偏好設定
  - [ ] 資料清除功能
  - [ ] 隱私提示

#### Phase 4: 增強功能
- [ ] **進階設定**
  - [ ] 分組抽獎（將參與者分為不同組別）
  - [ ] 權重設定（不同參與者有不同中獎機率）
  - [ ] 排除名單（某些人不參與此次抽獎）
  - [ ] 自訂抽獎標題

- [ ] **視覺效果**
  - [ ] 多種動畫主題選擇
  - [ ] 顏色主題切換（淺色/深色模式）
  - [ ] 自訂背景
  - [ ] 全螢幕模式

- [ ] **無障礙性**
  - [ ] 鍵盤導航支援
  - [ ] 螢幕閱讀器支援
  - [ ] 高對比度模式
  - [ ] 字體大小調整

#### Phase 5: 特殊功能
- [ ] **多語言支援**
  - [ ] 繁體中文（預設）
  - [ ] 英文
  - [ ] 簡體中文
  - [ ] 日文

- [ ] **分享功能**
  - [ ] 產生分享連結
  - [ ] 分享抽獎結果截圖
  - [ ] QR Code 產生

- [ ] **統計資訊**
  - [ ] 抽獎統計（總次數、參與人數等）
  - [ ] 中獎機率計算
  - [ ] 圖表顯示

## 技術需求

### 必要
- [x] HTML5
- [x] CSS3
- [x] Vanilla JavaScript (ES6+)

- [ ] File API (檔案上傳)

- [ ] 首次載入時間 < 1 秒

- [ ] 無 XSS 漏洞
- [ ] 無資料外洩
- [ ] 輸入驗證完整
- [ ] 不使用 eval() 或不安全的程式碼執行
- [ ] LocalStorage 資料加密（選用）
